# Hist-ria-interativa-para-deficientes-visuais
Histórias interativas para deficientes visuais no sentido que possa ser compreendias por eles.

Criado 3 histórias de forma que o usuário deficiente visual possa escolher e ouvir a história pelo site, tornando interativo e acessivel de certa forma. Ele vai poder escolher, pausar e ouvir as histórias.
